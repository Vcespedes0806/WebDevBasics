Margen, Relleno y Borde

CSS ofrece propiedades para controlar el espacio alrededor de los elementos, el margen (margin), y el espacio dentro de los elementos, el relleno (padding).
Se pueden definir bordes (border) alrededor de elementos, especificando el ancho, el estilo y el color del borde.

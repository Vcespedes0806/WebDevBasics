Reglas CSS: Propiedad, Valor y Selector

Una regla CSS está compuesta por tres partes: selector, propiedad y valor.
El selector apunta a los elementos HTML a los que se aplicará el estilo.
La propiedad especifica qué característica del elemento se va a cambiar (por ejemplo, color de fondo, tamaño de fuente).
El valor es el valor específico que se asigna a la propiedad (por ejemplo, un color hexadecimal, un tamaño de fuente en píxeles).

Tipos de Selectores

Los selectores por etiqueta aplican estilos a todos los elementos con la misma etiqueta HTML (por ejemplo, todos los párrafos <p>).
Los selectores por clase se utilizan para aplicar estilos a elementos con una clase específica (por ejemplo, <div class="encabezado">).
Los selectores por ID se utilizan para aplicar estilos a un elemento con un ID específico (por ejemplo, <div id="principal">).
Los selectores combinados permiten apuntar a elementos específicos en función de múltiples condiciones (por ejemplo, <p.intro.destacado>).

Comentarios en CSS

Los comentarios en CSS se escriben entre /* y */ y se utilizan para documentar el código y hacerlo más legible.
Los comentarios no afectan la apariencia de la página web y son útiles para anotar el propósito de ciertas reglas de estilo.
